== Installation ==

Using The WordPress Dashboard

1. Navigate to the 'Add New' Plugin Dashboard
2. Select `ti-wordpress-seo.zip` from your computer
3. Upload
4. Activate the plugin on the WordPress Plugin Dashboard

Using SFTP

1. Extract `ti-wordpress-seo.zip` to your computer
2. Upload the `ti-wordpress-seo.zip` directory to your `wp-content/plugins` directory
3. Activate the plugin on the WordPress Plugins Dashboard

Using composer

1. Run `composer require thoughtsideas/ti-wordpress-seo` on the command line
